# Gentle Space

A calming landing page built with Next.js, Tailwind CSS, and simple shadcn-style components.

## Start the project

1. Open a terminal in `c:\Users\User\Desktop\Project`
2. Run `npm install`
3. Run `npm run dev`
4. Open `http://localhost:3000`

## What this site includes

- Soft pastel palette and warm gradients
- Supportive, reassuring page copy
- Reusable UI components for buttons, cards, and typography
- Smooth, calming layout for an uplifting experience
